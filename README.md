# gwebmgr

#### 介绍
gps51.com 前端网页完整代码，主要功能：定位、管理设备、在线视频。

#### 软件架构
软件架构说明


#### 安装教程

1.  windows或者linux
2.  安装 java 1.7
3.  安装 tomcat7.0

#### 使用说明

1.  下载全部代码
2.  复制到 tomcat 的 webapps 目录下
3.  配置好 tomcat的端口 80 和 网站根目录
4.  启动tomcat
5.  输入ip地址测试是否出现网页登录界面
